# Техническое задание на разработку backend-системы CRM для массажного салона "РелаксПро"

## Общие сведения

Название: Система управления клиентами массажного салона "РелаксПро"

Цель проекта: Разработка серверной части веб-приложения для автоматизации и упрощения работы с клиентами массажного салона

Аудитория:
- Администраторы салона
- Массажисты
- Клиенты салона

## Функциональные требования

### Модуль управления аккаунтами

- Регистрация пользователей
- Управление ролями пользователей администратором
- Добавление дополнительной информации и изменение данных в профиле

### Модуль записи на массаж

- Отображение списка доступных слотов с заданными промежутками времени и специалистами
- Запись клиента на определенный слот с выбором услуги
- Отмена записи клиентом на определенный слот
- Подтверждение услуги массажистом
- Добавление и удаление временных слотов массажистами

### Модуль услуг и специалистов

- Предоставление данных об услугах и о специалистах
- Добавление новых услуг массажистами

### Модуль аналитики

- Формирование отчетов по выручке
- Формирование статистики по различным услугам

## Пользовательские роли

### Администратор

- Выдача ролей пользователям
- Получение аналитики по всей работе салона

### Массажист

- Просмотр своего расписания
- Редактирование своего расписания
- Подтверждение записей клиентов
- Получение аналитики по своей работе

### Клиент

- Управление своими записями на услуги

## Нефункциональные требования

### Безопасность

- Шифрование приватных данных
- Защита от базовых атак, например, sql injection

## Технические требования

### Архитектура

- Web-приложение с клиент-серверной архитектурой
- REST API

### Стек технологий

- Backend: Java Spring
- База данных: Postgresql
