# Техническое задание на разработку backend-системы CRM для массажного салона "РелаксПро"

## Общие сведения

Название: Система управления клиентами массажного салона "РелаксПро"

Цель проекта: Разработка серверной части веб-приложения для автоматизации и упрощения работы с клиентами массажного салона

Аудитория:
- Администраторы салона
- Массажисты
- Клиенты салона

## Функциональные требования

### Модуль управления аккаунтами

- Регистрация и авторизация пользователей
- Управление ролями пользователей администратором
- Добавление дополнительной информации и изменение данных в профиле

### Модуль записи на массаж

- Запись клиента на определенную услугу с выбором времени
- Отмена записи клиентом и массажистом
- Управление записью массажистом

### Модуль услуг и специалистов

- Предоставление данных о специалистах в системе пользователям
- Отображение списка доступных услуг от массажистов пользователю
- Добавление и удаление видов услуг администратором
- Добавление, удаление и редактирование своих услуг массажистами

### Модуль аналитики

- Формирование отчетов по выручке
- Формирование статистики по различным услугам

## Пользовательские роли

### Администратор

- Выдача ролей пользователям
- Получение аналитики по всей работе салона
- Доступ ко всему функционалу приложения

### Массажист

- Доступ к информации о конкретных пользователях в системе
- Редактирование списка предоставляемых услуг
- Управление записями клиентов на услуги массажиста
- Получение аналитики по своей работе

### Клиент

- Запись на услуги массажистов
- Отмена своих записей
- Доступ к просмотру и редактированию своего профиля

## Нефункциональные требования

### Безопасность

- Шифрование приватных данных
- Защита от базовых атак, например, sql injection

## Технические требования

### Архитектура

- Web-приложение с клиент-серверной архитектурой
- REST API

### Стек технологий

- Backend: Java Spring
- База данных: Postgresql
